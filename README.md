# rvizer
Modern web-based 3D scene visualizer for robotics project.

The main purpose of this project is for general purpose of robotics visualization.
- Just Visualization
- No Simulation
- Fast Prototyping Robotics Scene

Package consists of mini-packages such as
- Bubblify: Approximate Robot with Sphere
- Composer: Compose Multiple URDF Into One (Robot Arm + Gripper)
- Rvizer: Full Scene Visualizer (Robot Arm + Environment + Robot Ghost)

#### Dependencies
```
viser v@1.0.30
yourdfpy v@0.0.60
robot_descriptions v@1.21.0
```
